# schoolfootballoffice

> A Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
``` 
├── build                      // webpack 等相关配置
├── config                     // 打包环境相关配置
├── src                        // 源代码

│   ├── assets                 // 主题 字体等静态资源

│   ├── styles                 // 全局样式
```
